# Proyecto_Final

Simulación de un robot planar de 3 grados de libertad (3 GDL) utilizando Simulink, Simscape Multibody y Simscape Electrical.

El proyecto integra:
- Modelado mecánico del robot en SolidWorks.
- Motores de corriente directa (DC) en cada articulación.
- Convertidor Buck como etapa de alimentación.
- Puente H para el control del motor.
- Control PID en lazo cerrado para regular la posición angular.

Además, se analizaron variables como posición, velocidad angular, corriente, voltaje y señales de control mediante diferentes scopes y gráficas.

Desarrollado como proyecto final de la materia de Modelado y Simulación de Sistemas Dinámicos.
